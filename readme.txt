Описание приложения

source - папка на которую настривается веб-серверъ
config.php - конфиг БД


1. Развертывание веб-части: copy папку source на вебсервер, кнфигурация параметров БД, развертывание БД (DB_WEB), 
2. Компиляция моб приложения: zip.bat, затем получившийся project.zip оптравляем на build.phonegap.com, 
оттуда скачиваем apk приложения
3. enjoy!
